AWS-Dev

Live, hands-on workshop. Build and deploy a real serverless app using IAM, S3, DynamoDB, Lambda, Step Functions & more.

Key Takeaways?

Day 1 – Foundations & Core Services

  • Module 1: Kickoff & Introduction
    Overview of the workshop, serverless concepts, and how AWS enables rapid application development.
  • Module 2: IAM – Users, Roles & Policies
    Deep dive into IAM fundamentals, permissions, and security best practices. Hands-on: create secure IAM roles.
  • Module 3: Amazon S3 – Storage Backbone
    Buckets, policies, encryption, and lifecycle management. Hands-on: host a static website with S3 and enforce access controls.
  • Module 4: DynamoDB – Serverless Database
    NoSQL design principles, partition keys, indexes, and scalability. Hands-on: build a table and run queries.
  • Module 5: Mini Project
    Combine S3, DynamoDB, and IAM into a working mini application to practice integration of core services.

Day 2 – Serverless Compute & Orchestration

  • Module 6: AWS Lambda – Core Compute
    Function lifecycle, triggers, environment variables, and monitoring. Hands-on: deploy Lambda functions and connect them to S3/DynamoDB.
  • Module 7: Messaging with SQS & SNS
    Event-driven architectures with queues and pub-sub. Hands-on: build a workflow that triggers Lambda functions via SQS/SNS.
  • Module 8: AWS Step Functions – Orchestration
    State machines, branching, and error handling. Hands-on: create a workflow that coordinates multiple Lambda functions.
  • Module 9: Monitoring (CloudWatch & X-Ray)
    Set up CloudWatch metrics, logs, and dashboards. Use X-Ray for tracing and debugging distributed workflows.
  • Module 10: Capstone Project
    End-to-end project using IAM, S3, DynamoDB, Lambda, Step Functions, SNS/SQS. Deploy and test a fully functional serverless application.
